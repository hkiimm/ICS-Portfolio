ARRAYS AND ARRAYLISTS



ARRAYS



Arrays are used to store multiple pieces of information, of the same data type, in one place. Data types can be primitive, such as an int or double, or it can be an object. For example, an array can contain a list of student numbers. The index of an array ranges from arr[0] to arr[arr.length - 1]. If at any point, you try to access elements that are not within the arrays length, Java will throw an ArrayIndexOutOfBoundsExcpetion.



INITIALIZATION

Since arrays are objects in Java, it requires the keyword new when an empty array is being declared. When a value is declared within the second pair of square brackets, it determines the length of the array. The length of the array is set as a constant and cannot be modified, therefore no more than the set number of elements can be stored within the array. An empty array will set int and double to 0, boolean to false, and objects will reference to null. It can be declared as such: 


          
            int [] arr = new int [10];

To set values to the elements within an empty array, you need to use the arrays name followed by a set of square brackets, and the index of the elements location. You can choose to set values to only some indexes, and not others. It can be declared as such: 


          
            int [] arr = new int [10];
            arr[0] = 5;
            arr[1] = 7;

An array can also be initialized with a set of values using curly braces and commas. It can be declared as such: 


          
            int [] arr = {1, 2, 3, 4, 5};

ACCESSING ELEMENTS

Arrays can be accessed by using the name of the array, and square brackets with the index of the element. It would look like this: 


          
            arr[i]
            arr[0]

When you need to access all the elements within the array, you can use a for loop or a for-each loop. A for-each loop is used to access the elements within the array, when you don't need to modify or set any values within the array. For-each loops also do not keep track of the index. 


          
            int i = 0;
            for (int num : arr) {
              if (num == i)
                i++;
              return i;
            }

A for loop is used to access the index of the element and can modify the values of the elements. When traversing an array, the for loop runs from arr[0] to arr[arr.length - 1]. The index i starts at 0, the first element, and then continues to increase until it has gone through the entire array. 


          
            for (int i = 0 ; i < arr.length ; i ++) {
              arr[i] = i + 1;
            }


TWO-DIMENSIONAL ARRAYS



INITIALIZATION

Just like one-dimensional arrays, two-dimensional arrays can be initialized using the keyword new, or with a set of values. Creating a new 2D array is quite similar to creating a 1D array, except you now have 2 pairs of square brackets. It can be declared as such: 


          
            int [][] arr = new int [5][5];

Once again like 1D arrays, 2D arrays can be initialized with values using curly braces and commas. It can be declared as such: 


          
            int [][] arr = {{1, 2, 3}, {1, 2, 3}, {1, 2, 3}};

ACCESSING ELEMENTS

When you're accessing elements of 2D arrays, you must specify the index of the row and column you're looking for. Instead of using one set of square brackets, you use two pairs with the indices you would like to access. It would look like this: 


          
            arr [i][x]
            arr [0][1]

Traversing two-dimensional arrays now uses two for loops instead of one, because we need to access the index of the row and column. Instead, you cannot use arr.length to find the length of the array. Now, arr.length returns the number of rows, not the number of elements. To find the number of columns, or number of elements in that row, you can use row[0].length. The code example below is how you would traverse a 2D array using for loops. 


          
            //traverses the rows
            for (int i = 0 ; i < arr.length ; i++) {
              //traverses the columns
              for (int x = 0 ; x < arr[i].length ; x++) {
                System.out.println (arr[i][x] + " ");
              }
            }


ARRAYLISTS



The arraylist class from the java.util library is another way to store objects. Arraylists have the advantage of being able to modify the length to fit the elements as needed. Arraylists are very dynamic, which allows them to grow and shrink appropriately. They can add and remove elements by implementing the List < E > interface. The following imports should be made: 


          
            import java.util.ArrayList;
            import java.util.List;

or


          
            import java.util.*;

INITIALIZATION

Just like arrays, arraylists require the keyword new because it is a class. Unlike arrays, you cannot set the values of an arraylist while you're declaring it. Primitive data types such as int, double, char, boolean cannot be stored in arraylists. To declare an arraylist, you must use the < > symbols. It can be declared as such: 


          
            Arraylist < Integer > arr = new ArrayList < Integer > ();

LIST< E > INTERFACE

A list of type E elements are held in the List < E > Interface. The List < E > Interface is important to arraylists because it provides the functions to access and modify the elements within the list. The most frequently used methods retrieved from the Barron's textbook are:

  • boolean add (E element) - adds the element to the end of the list
  • void add (int index, E element) - adds the element at the index
  • E remove (int index) - removes the element at the index
  • E get (int index) - returns the element at the index
  • E set (int index, E element) - replaces the element at the index, with the new element
  • int size () - returns the number of elements in the list

ACCESSING ELEMENTS

Arraylists can be accessed using both a for loop or a for-each loop. Below is an example of traversing an arraylist using a for loop. 


          
            for (int i = 0 ; i < arr.size() ; i++){
              if (arr.get(i).equals(num))
                return true;
            }
            return false;

Arraylists can be accessed using for-each loops as well, as shown below: 


          
            ArrayList< Integer > arr = new ArrayList< Integer >();
            //adds elements to the arraylist
            arr.add(1);
            arr.add(2):
            arr.add(3);
            //uses a for loop to display contents of the list
            for (String s : arr)
              System.out.println (s);
